Estructura base

Documento HTML5

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport"
    content="width=device-width, initial-scale=1.0">

07

Box Model

Reset recomendado

*, *::before, *::after {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

border-box: el width incluye padding y borde.

Espaciado y bordes

margin: 10px 20px; // vert horiz
padding: 8px 16px 8px 16px;
  // arriba der abajo izq
border: 2px solid #000;
border-radius: 12px;
outline: 2px dashed red;
box-shadow: 0 4px 12px rgba(0,0,0,.2);

Display

blockOcupa todo el ancho
inlineEn línea, sin width/height
inline-blockEn línea pero dimensionable
flexContenedor flexible
gridContenedor de rejilla
noneOculta (no ocupa espacio)
08

Flexbox

Contenedor padre

.flex {
  display: flex;
  flex-direction: row | column;
  justify-content: center;
    // eje principal
  align-items: center;
    // eje cruzado
  flex-wrap: wrap;
  gap: 16px;
}

justify-content

flex-startAl inicio
centerCentrado
flex-endAl final
space-betweenExtremos + huecos iguales
space-aroundEspacio alrededor
space-evenlyEspacio uniforme

Items hijos

.item {
  flex-grow: 1;   // crecer
  flex-shrink: 0; // encoger
  flex-basis: 200px;
  flex: 1 1 auto; // abreviado
  align-self: flex-end;
  order: 2;
}
09

CSS Grid

Definir la rejilla

.grid {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto 1fr;
  gap: 20px;
}

Columnas automáticas

// rejilla responsive sin media query
grid-template-columns:
  repeat(auto-fill,
    minmax(200px, 1fr));

// repetir N columnas iguales
grid-template-columns: repeat(3, 1fr);

Posicionar items

.item {
  grid-column: 1 / 3;  // de línea 1 a 3
  grid-row: span 2;    // ocupa 2 filas
  grid-column: 1 / -1; // todo el ancho
}

.grid {
  place-items: center; // centra todo
}

Posicionamiento

position

staticPor defecto, flujo normal
relativeRelativo a su posición original
absoluteRelativo al ancestro posicionado
fixedRelativo a la ventana
stickyPegajoso al hacer scroll

Ejemplos

.fijo {
  position: fixed;
  top: 0; left: 0;
  z-index: 100;
}
.pegajoso {
  position: sticky;
  top: 20px;
}
// centrar absoluto
top: 50%; left: 50%;
transform: translate(-50%, -50%);

13

Transiciones y animaciones

Transition

.btn {
  transition: all .3s ease;
  // prop duración timing delay
  transition: transform .2s
    ease-in-out .1s;
}
.btn:hover {
  transform: scale(1.05);
}

Transform

transform: translate(10px, 20px);
transform: scale(1.2);
transform: rotate(45deg);
transform: skew(10deg);
// combinadas
transform: translateY(-4px)
  rotate(3deg) scale(1.1);

Keyframes

@keyframes aparecer {
  from { opacity: 0; }
  to   { opacity: 1; }
}
.elem {
  animation: aparecer 1s ease
    infinite alternate;
}

Responsive

Media queries

// móvil primero
@media (min-width: 768px) {
  .grid { grid-template-columns: 1fr 1fr; }
}
@media (max-width: 600px) {
  .nav { display: none; }
}
@media (prefers-color-scheme: dark) {
  body { background: #000; }
}

}

  margin-inline: auto;
  padding-inline: 1rem;
}

  aspect-ratio: 16 / 9;
  width: 100%;
}

Scroll suave

html {
  scroll-behavior: smooth;
}

}
